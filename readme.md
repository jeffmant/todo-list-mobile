## Key Features

* Create Task Item
* Check Task as Done
* Check Task as Todo
* Delete Task
* Todo Tasks Count
* Done Tasks Count

## How To Use

```bash
# Install dependencies
$ npm install

# Run the app
$ npx expo start
```
